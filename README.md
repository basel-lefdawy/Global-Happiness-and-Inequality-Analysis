# 🌍 Global Happiness Analysis

## 📊 Overview
This project analyzes the relationship between economic performance, income inequality, and happiness across countries.

## ❓ Research Question
Does income inequality affect happiness even in countries with high GDP per capita?

## 📁 Datasets Used
- World Happiness Report
- GDP per capita (World Bank)
- Gini Index (World Bank)

## 🧹 Data Wrangling
- Removed unnecessary columns
- Reshaped datasets using melt
- Handled missing values
- Merged datasets into one unified dataset

## 📈 Key Findings
- GDP has a strong positive relationship with happiness
- Income inequality negatively affects happiness
- Economic wealth alone is not enough to ensure well-being

## 🛠️ Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## 🚀 Conclusion
Both economic performance and fair income distribution are important factors in determining happiness levels.
